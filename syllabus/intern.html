Data Cleaning

Most of the publically available data are not in scientific form. We may need to use some software of manual approach to generate the scientific data. For a single person, it is not possible to perform this task. But we will divide the job to multiple participants to make the data in the required form for our models.

Data Visualization

Data visualization is part of exploratory data analysis. You may need some knowledge of modern algebra and statistics. We can use Python, R, javascript, Plotly, Tablue, etc for data visualization.

Network design and Analysis

You can participate in network analysis using python NetworkX library or Neo4j with network database. Networks can also be visualized by utilizing both javascript and neo4j browser.

Machine Learning Model Development

After getting familiar with the fundamentals of Machine Learning from Intermediate level, you can start to build models using scikit-learn library. Some models also are based on Bayesian Learning.

Database Development

A database is inherently connected with big data manipulation. We will use MongoDB, SQL and Django/Flask approach to handling the database. Neo4J has its graphical database, or it can work along with Mangodb or SQL interface.

BigData Tools

Hadoop ecosystem is a massive platform in BigData. We can include some exciting components based on our interest. We will use Elasticsearch for indexing and search purpose. Spark for key-value pair generation, and map reduce.

Deep Learning

Deep Learning is itself a huge concept including Convolutional Neural Network (CNN), Reinforcement Learning (RNN), GANs etc. There are several opportunities to implement deep learning models in network and complex system.
