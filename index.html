Toy Model of Corruption Detection

This is a toy model over a simulated data under a typical bureaucratic and political scenario. The model will use the method of Deep learning (Machine learning) and network embedding method to train over the data and produce a prediction algorithm. THis model is completed in two part.
  • In the first part, the data simulator will create the hypothetical network data and the model will systematically study(training a model) the networking and activities of the member up to the nth degree(based on computational power available) of relationship in the graph. For example, a politician's relative/friends get an abrupt increase in property value will add the probability to the politician be corrupted. THis model assumes that all member's property is accounted at the beginning and runs the simulation for next 10 years where as in the runtime the algorithm is built.
  • In the second run of the simulation, them model accuracy will be tested. It will start the detection of corrupted persion from the simulated network.
